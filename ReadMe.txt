3.Usage
*********************************************************************
the applcation rquires a login from the user using email and a
password, if you have not registered you can register and login after
The application prompts the user for input into textboxes in the 
windows. The application asks the user to enter Module Name,
Module code, module credits ,semester start date (via calendar),
number of weeks in the semester and number of hours spent in class for 
the module. 
The User then can select diffrent functions to update study time,start
a new week of study and view a graph by clicking on the buttons to do 
so for the specific module on the view modules page.
